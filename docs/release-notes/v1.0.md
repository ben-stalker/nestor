# Nestor v1.0.0 Release Notes

**Released:** 2026-05-18

Nestor v1.0.0 is the first stable release — a self-hosted household dashboard designed for a
touchscreen kiosk in your home.

## Highlights

- **Complete family hub** — profiles with PIN protection, chores & rewards, baby tracking,
  health logs, children's routines, mood log, NHS vaccination schedule.
- **Full calendar** — Google, Apple, and Yahoo CalDAV sync; day, week, and month views;
  recurring events; custody schedule; term dates.
- **Meal planning** — recipe library with URL import (Schema.org), 7-day meal planner,
  shopping list.
- **Home management** — bin schedules with UK bank-holiday awareness, subscriptions, home
  maintenance log, meter readings, budget tracking, checklists.
- **Finance tracker** — agreements (PCP/mortgage), savings goals, regular commitments,
  debt paydown chart.
- **Pets & vehicles** — pet health logs, vet visit calendar, vehicle bookings, fuel log,
  business mileage PDF export.
- **Voice commands** — wake word → Whisper STT → Piper TTS pipeline with quiet hours.
- **Plugin system** — community plugins via manifest, error-isolated loading, home-screen widgets.
- **Setup wizard** — 10-step first-run wizard with language, locale, profiles, calendars,
  display, orientation, voice, features, plugins, and done steps.
- **Internationalisation** — i18next with English + French, locale-aware format helpers, RTL prep,
  colour-blind palettes.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/nestorapp/nestor/main/install/install.sh | sudo bash
```

Tested on Ubuntu 22.04 LTS and 24.04 LTS.

## Upgrading

```bash
sudo systemctl stop nestor-server
git -C /opt/nestor pull
npm --prefix /opt/nestor ci
npm --prefix /opt/nestor run build
sudo systemctl start nestor-server
```

Or use **Admin → System → Check for Updates** in the UI.
